# passive-portfolio-lab
A Buy & Hold backtesting platform that explores the historical performance of different risk profiles and estimates the time needed to achieve FIRE.

## Overview
This project helps everyday investors who do not actively trade understand the historical performance of a Buy & Hold strategy across four risk profiles: Low Risk, Medium Risk, High Risk, and Crypto Believer.

## Core Question
When a portfolio's backtested return is higher, what is the accompanying cost?

## Tech Stack
- Data collection: yfinance, FRED API
- Data processing: Python, pandas
- Data storage: BigQuery
- Frontend: Streamlit
- Additional dashboard: Looker Studio
